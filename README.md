# AL HASIB - Futuristic AI Portfolio

একটি আধুনিক এবং ভবিষ্যতমুখী পোর্টফোলিও ওয়েবসাইট যা AI এর শক্তি প্রদর্শন করে।

## 🚀 বৈশিষ্ট্য

✨ **আল্ট্রা-মডার্ন UI**
- ডার্ক থিম with নিওন নীল এবং বেগুনি অ্যাকসেন্ট
- গ্লাসমরফিজম এফেক্ট
- মসৃণ অ্যানিমেশন
- সাইবারপাঙ্ক-অনুপ্রাণিত ভিজুয়াল

🤖 **AI সহায়ক**
- বিল্ট-ইন চ্যাটবট
- ভয়েস ইন্টারঅ্যাকশন সাপোর্ট
- রিয়েল-টাইম প্রতিক্রিয়া
- আধুনিক চ্যাট ইন্টারফেস

📱 **সম্পূর্ণ প্রতিক্রিয়াশীল**
- মোবাইল, ট্যাবলেট এবং ডেস্কটপের জন্য অপ্টিমাইজ করা
- প্রিমিয়াম স্টার্টআপ-মানের ডিজাইন

## 📋 সেকশন

1. **হিরো সেকশন** - প্রোফেশনাল প্রোফাইল ইমেজ এবং CTA বাটন
2. **আমার সম্পর্কে** - ব্যক্তিগত পরিচয় এবং দক্ষতার সংক্ষিপ্ত বিবরণ
3. **দক্ষতা** - AI, Next.js, React, TypeScript, Python ইত্যাদি
4. **প্রজেক্ট** - শোকেস এবং ফিল্টারিং সহ
5. **অভিজ্ঞতা** - টাইমলাইন এবং পুরস্কার
6. **যোগাযোগ** - ফর্ম এবং সোশ্যাল লিংক
7. **AI সহায়ক** - চ্যাটবট এবং ভয়েস সাপোর্ট
8. **ফুটার** - দ্রুত নেভিগেশন

## 🛠️ প্রযুক্তি

- **Next.js 15** - আধুনিক React ফ্রেমওয়ার্ক
- **TypeScript** - টাইপ নিরাপত্তা
- **Tailwind CSS** - ইউটিলিটি-ভিত্তিক স্টাইলিং
- **Framer Motion** - মসৃণ অ্যানিমেশন
- **React Icons** - আইকন লাইব্রেরি

## 📦 ইনস্টলেশন

```bash
# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# ডেভেলপমেন্ট সার্ভার চালান
npm run dev

# ব্রাউজার খুলুন এবং যান
# http://localhost:3000
```
